Kickabout Messenger Bot

Created a messenger bot which allows people to find and organise football games. It also allows booking a game by paying through credit card using the Stripe API. The bot also uses natural language processing, powered by Wit.ai, to develop a conversational tone and interact with the user.
I made this from scratch during my summer internship at Activate Technologies. Used Node and Express to write the server for the bot and MongoDB database for handling users, games and analytics.

Facebook Messenger API

Stripe API

I was also tasked with creating an analytics dashboard. I used D3.js to visualize the data and wrote custom functions to track interactions and store them in the MongoDB.

D3.js

Mongoose.js

I also created a user accounts system indexed by the Messenger App Scoped IDs and also used the Facebook SDK and OAuth to connect with the main Facebook ID's. This also helped with the payments system, implemented using the Stripe API, by allowing us to get information about and keep track of the people attending the games. I also created a secure admin interface for the company to use. The admin interface had features like a CMS for other team members to add games to the bot. I used Amazon S3 to handle images.
